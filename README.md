# FocusFlow: Pomodoro Timer Extension

A sleek, dark-themed Chrome extension designed to boost productivity using the Pomodoro Technique. It allows users to manage work sessions, track daily progress, and maintain a task checklist directly from their browser.

## Features

* **Customizable Timer**: Set your preferred durations for work sessions, short breaks, and long breaks.
* **Live Statistics**: Track the number of completed Pomodoros and total focus time for the day.
* **Integrated Task Checklist**: Add and manage tasks within the extension to stay focused on your current objectives.
* **Dark Mode UI**: A modern, aesthetically pleasing dark interface designed to reduce eye strain.
* **Session Management**: Quick-action buttons to toggle between work, short breaks, and long breaks.

## Installation

1. Clone this repository or download the source code.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** by toggling the switch in the top right corner.
4. Click **Load unpacked** and select the directory containing the extension files.

## Usage

1. Click the FocusFlow icon in your Chrome toolbar.
2. Use the **Customization** section to set your timer intervals.
3. Add your goals to the **Task Checklist**.
4. Press **Start** to begin your focus session.
5. Review your progress in the **Today's Stats** section.

## Technologies Used

* HTML5
* CSS3 (Modern UI/UX design)
* JavaScript (Web Extension API)
